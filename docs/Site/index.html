Scribble is a programming language which can be embedded into a larger application to allow customization of easy extension of existing functionality.

Variables & Types

A slightly different approach is taken to variable decleration.

For starters Scribble uses a right handed type decleration, what this means in practice is that you give your variables their type on the right of the declaration not the left. For example to declare the variable i to be of type integer you would write var i : int := 0; instead of int i := 0; in a language which uses a left handed system.

As the previous example may have suggested, Scribble moves away from using = for variable assignment and instead uses := to handle assignment.

To avoid excessively verbose code, you can ask the compiler to infer ( work out ) the type of a variable when it is declared by looking at what it is being assigned too. For example to declare a variable i and infer its type to be int you can write var i := 0; the compiler would then look at the expression on the right hand side of the decleration and would use the type of that to decide that variable i was an integer. Note: The compiler cannot infer type if you are not assigning the right hand side so var j : int; is valid and var j := 0; is valid but var j; is not.

Methods

Scribble programs are procedural, built up around sets of methods.


func Fibs(n : int) : int {
 if n = 1 then
  return 0;
 if n = 1 then
  return 1;
 return Fibs(n-1) + Fibs(n-2);
}

func main() {
 var j := Fibs(30);
}

Packages

Every file is seen as a package, a package is comprised of methods and structures.


One example of this would be using a method inside the math library to return the absolute value of an integer.


package math := import("math/math");

func main() {
 var i := math.Abs(-15);
}
