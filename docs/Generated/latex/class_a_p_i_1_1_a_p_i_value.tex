\subsection{Detailed Description}
The \hyperlink{class_a_p_i_1_1_a_p_i_value}{A\-P\-I\-Value} class contains the arguments for or return value from a virtual machine function.

It is a wrapper around the register value and heap data to provide an easy way to extra useful data from the result of a V\-M execution or pass information to it.
