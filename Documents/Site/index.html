Scribble is a programming language which can be embedded into a larger application to allow customization of easy extension of existing functionality.

Variables & Types

A slightly different approach is taken to variable decleration.

For starters Scribble uses a right handed type decleration, what this means in practice is that you give your variables their type on the right of the declaration not the left. For example to declare the variable i to be of type integer you would write var i : int := 0; instead of int i := 0; in a language which uses a left handed system.

As the previous example may have suggested, Scribble moves away from using = for variable assignment and instead uses := to handle assignment.

To avoid excessively verbose code, you can ask the compiler to infer ( work out ) the type of a variable when it is declared by looking at what it is being assigned too. For example to declare a variable i and infer its type to be int you can write var i := 0; the compiler would then look at the expression on the right hand side of the decleration and would use the type of that to decide that variable i was an integer. Note: The compiler cannot infer type if you are not assigning the right hand side so var j : int; is valid and var j := 0; is valid but var j; is not.

Flow Control

If it ain't broke.

Scribble has flow control similar to most conventional languages. It offers flow control through if statements, for loops and while loops.

If

If statements are executed if a given boolean expression evaluates to true in the form if Condition then { Code } For example if true then { Do Something. } will always execute the code in it. They can be chained together using the else keyword for example if someExression then { Code. } else if someExpression then { Else code. }

For

For loops in written take an initializer, a condition and a step. They are written in the form for Initial; Condition; Step do { Code }. The loop will keep executing until the condition expression is false.

An example of a for loop would be
for var i := 0; i < 100; i++ do {
 sys.Write("Hello World");
}

While

While loops continously execute their body until a condition is false, they are written like while Expression do { Code. }

An example: var i := 0;

while i < 100 do {
 i++;
}

Arrays

Arrays & How they work

All arrays in Scribble reside in a garbage collected heap. The heap uses a deferred garbage collection routine to periodicly delete any memory that is no longer being used. An arrays type is written as array(IndexType), for instance to create a nil reference to an array of integers you would write var j : array(int);. The variable j could then be used as a reference to an array of integers on the heap.

When you define a variable of an array type ( var j : array(Anything) ) it is initialized as a nil reference ( The equivilent in C would be a null pointer ) to actually create an array you use the [NumberOfElements]BaseType syntax. For example to create an array of 100 integers you would write var j := [100]int;. The number of elements specified has to be an integer expression

To access an element within the array the [IndexNumber] syntax is used.

The len(Array) function will return the number of elements in an array. E.g. len([100]int) = 100

An example of creating an array of integers and assigning each element to be the numeric value of its index and then printing it out to the screen.

var j := [100]int;
for var i := 0; i < len(j); i++ do {
 j[i] := i;
 sys.Write(sys.String(j[i]));
}

Functions

Scribble programs are procedural, built up around sets of functions.

Functions can be defined multiple times for different inputs. The compiler will select the correct one to use at runtime from the types of the arguments ( See Abs or Diff in examples/math/math.sc for a good example of this ). If a function is not defined with an explicit return type then is assumed to be of type Void ( That it does not return any value ).


An example of a function to compute the n'th member of the fibonacci sequence.

func Fibs(n : int) : int {
 if n = 1 then
  return 0;
 if n = 1 then
  return 1;
 return Fibs(n-1) + Fibs(n-2);
}

func main() {
 var j := Fibs(30);
}

Packages

Every file is seen as a package, a package is comprised of methods and structures.


One example of this would be using a method inside the math library to return the absolute value of an integer.


package math := import("math/math");

func main() {
 var i := math.Abs(-15);
}
